# ICIO Networks

<img src="diagrams/icioNet_menu.png" alt="ICIO Networks Menu"/>

The application source code is available on
[GitHub](https://github.com/bowerth/desk/blob/master/inst/industry/tools/indic/icioNet.R)

Network visualisations
:   Application to create network diagrams from ICIO indicators. The charts are using [d3.js](d3js.org) and were created with the [networkD3](https://github.com/christophergandrud/networkD3/) infrastracture and [htmlwidgets](https://github.com/ramnathv/htmlwidgets).

## Cluster Network

Hover over a country to highlight important trade partners. Position of countries on the circle according to world region.

blue line
:   other countries import from selected country

red line
:   top five countries that the selected country imports from

See data at bottom of page

Source: http://bl.ocks.org/mbostock/7607999

## Treemap Network

Size of tiles represents countries' total exports

Arcs represent important trade flows from a country perspective

See data at bottom of page

## Chord Network

Chords represent bi-directional trade flows. Hover over ribbon to see tooltip with numbers or on the country at the edge for total size.
